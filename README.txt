Main Interface
Open CSV: Click to open a CSV file. A file dialog will appear for you to select your CSV file.
Save CSV: After making changes, click this to save your CSV file.
Columns Listbox: This displays all the columns from your CSV. You can select multiple columns by holding Ctrl/Command for specific actions.

Editing Columns
New Column Name: Enter the name for a new column here.
Add: Adds the new column to the CSV with random data.
Remove: Removes the selected columns from the CSV.

Filling Data
Custom Random Fill: 
Select columns, then click this button. A dialog will let you choose how to fill:
Numeric: Fills with random numbers.
Alphabetic: Fills with random uppercase letters.
Alphanumeric: Fills with a mix of letters and numbers.
Specific Fill:
Select columns, then click this to fill with a specific value you provide in a prompt.
Email Fill:
Select a column for email addresses.
Enter an email user ID and custom domain in the dialog that appears.
Choose whether to append a random alphanumeric or numeric value after the user ID.
Emails are generated in the format userID+randomValue@domain.com.

Notes
When adding a new column, if no data exists, 100 rows will be created. 
Ensure you save your work before closing the application or opening another file to avoid losing changes.

Features
Open and Save CSV files
Add or remove columns
Fill columns with random or specific data
Generate email addresses with customizable domain
